# openclaw_avatar - 虚拟形象桌面程序

## 项目概述

一个运行在桌面上的虚拟形象，可以与用户进行语音和文字对话。

## 功能需求

1. **虚拟形象展示** - 卡通形象，有动画效果
2. **语音对话** - 用户说话，程序语音回复
3. **文字聊天** - 支持文字输入输出
4. **表情动画** - 根据状态显示不同表情（待机、说话、思考等）
5. **窗口控制** - 最小化、关闭等基本功能

## 技术栈

- **桌面框架**: Electron + React
- **语音识别**: Web Speech API
- **语音合成**: Web Speech API (TTS)
- **AI 对话**: 通过 ACP 连接 Kimi

## 项目结构

```
openclaw_avatar/
├── electron/          # Electron 主进程
├── src/               # React 前端源码
├── public/           # 静态资源
└── package.json      # 项目配置
```
